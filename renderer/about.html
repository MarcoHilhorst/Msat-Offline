Instructions

Put your data into a table with the same headings as below (replace m1, m2 etc with your own marker names). Add more columns or rows as needed, but ensure that your data are within the table otherwise the program won't read it.

For each sample, set the value of the Dropout column to either TRUE or FALSE. Setting it to true will allow a homozygous genotype to match with a heterozygous one provided one allele is shared between the two.

Finally, you will need to name the sheet that contains your tabulated data "M-sat". Note - not the actual excel document but just the worksheet tab down the bottom of the screen.
